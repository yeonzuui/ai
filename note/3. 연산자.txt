 ① 우선순위에 따른 연산자 종류:
 ② 1차 연산자: () 를 사용해 우선순위 변경 가능
 ③ 단항 연산자: ! (NOT), ++ (1씩 증가), -- (1씩 감소)
 ④ 이항연산자 - 승법연산자: * (곱셈), / (나눗셈), % (나머지)
 ⑤                   - 가법연산자: + (덧셈), - (뺄셈)
 ⑥                   - 관계연산자: <   <=   >   >=   ==   != (두 값 비교)
 ⑦                   - 비트곱연산자:  &
 ⑧                   - 비트합연산자:  |
 ⑨                   - 논리곱연산자:  &&
 ⑩                   - 논리합연산자:  ||
 ⑪ 조건연산자(3항연산자): 조건  ?  True  :  False
 ⑫ 할당연산자:  =   +=   -+   *=   /=   %=
