// Collection: 배열과 비슷한 자료구조
// 가장 많이 쓰는 ArrayList(List 구조), HashMap(Map 구조)

// int ArrayList 못 만듦 → integer 넣으면 가능 (*double도 Double로 만들면 가능) 
// ArrayList에서는 List와 달리 인덱스 추가 제한 없음
// add(): 인덱스 추가
// set(idx, "변경값"): 인덱스 수정
// size(): 길이
// get(): 인덱스값 반환 
// remove(n): n번째 인덱스값 삭제(n번 인덱스 → n-1번 인덱스)
// isEmpty(): 데이터가 존재하는지 확인(T/F)
// clear(): 자료구조 내 모든 데이터 제거

// HashMap<K, V> key값 type(object만 가능), 인덱스값 type
// Iterator<key값 type> : 반복자
// keySet(): 키값 차례대로 정렬
// hasNext(): 다음 데이터 존재 여부(T/F)
