 ① 제어문: 위에서 아래로 실행되는 코드 흐름을 원하는 방향으로 제어할 수 있게 함
 ② 반복문: 특정 코드를 여러 번 실행할 때 사용
 ③ for문: 
     for( 초기값 ; 반복 조건; 증감식 ) { 
              반복할 명령문들 };
 ④ while문: 
     while (조건식) { 
              조건이 true일 동안 계속 반복할 명령문들 };
 ⑤ do-while문: 
     do { 
              최소 한 번 실행, 두 번째부터는 조건식이 true일 동안 계속  반복할 명령문들 
     } while (조건식);
 ⑥ for는 반복 횟수가 정해져 있을 때, while은 조건에 따라 반복할 때 유용.
 ⑦ for는 정해진 횟수만큼 반복하고, while은 조건이 T일 경우에 반복하다가 F가 되면 반복을 멈추기 때문
 ⑧ break: 반복을 멈추고 반복문을 빠져나감
 ⑨ 반복문은 코드양이 많아지는 것을 방지, 효율적인 코드 작성에 필수적.
