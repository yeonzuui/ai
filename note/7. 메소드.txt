 ① 메소드(method, 함수): 특정 작업을 수행하는 로직
 ② 메소드 문법: 
      접근제한자 [static] 리턴type 메소드명(매개변수1, 매개변수2, ..) { 
                  실행할 코드들;
      	[return 리턴값];
       }
 ③ 접근제한자: public, protected, default, private 순으로 접근 제한이 강화
 ④ static: 메소드가 포함된 클래스에 고정적으로 위치(클래스 메모리에 저장됨), 따라서 인스턴스 변수를 만들지 않아도 다른 package에서 비교적 편리하게 호출이 가능해짐.
 ⑤ 리턴type: 메소드가 실행되어 전달하는 결과값의 타입, 리턴값이 없으면 void로 작성
 ⑥ 메소드명: 소문자로 시작하고 캐멀 스타일로 작성 ex) getName()
 ⑦ 매개값이 없다면 매개변수는 생략 가능
 ⑧ overloading: 같은 이름의 메소드를 매개변수를 달리하여 여러 개 정의
 ⑨ 메소드 호출: double result = divide(5,3);
 ⑩ 메소드 필요성: 동일 작업 반복, 유지보수에 용이
