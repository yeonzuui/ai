 ① 조건문은 특정 조건에 따라 코드를 실행할 때 사용
 ② 조건을 잘못 걸면 무한 loop에 걸리니 주의
 ③  if문: 
       if ( 조건식 1 ) { 
                 조건식 1이 true일 때 실행할 명령문;
       }else if ( 조건식 2 ) {
                 조건식 1이 false, 조건식 2가 true일 때 실행할 명령문;
       }else {
                 위의 조건식 둘 다 참이 아닐 때 실행할 명령문;
       }
 ④ if문만 써도 무방하나, 여러 조건을 처리할 때는 else if문 사용
 ⑤ switch문:
      switch ( 변수 ) { 
      case 변수값1: 
                 변수값1일 때 실행할 명령문; break;
      case 변수값2:
                 변수값2일 때 실행할 명령문; break;
      default: 
                 위의 변수값들이 아닐 때 실행할 명령문; break;
      }
 ⑥ switch문은 값이 특정 범주에 속할 때 가독성 면에서 좋음
 ⑦ 삼항 연산자(조건 ? 참 : 거짓): 간단한 조건을 처리할 때 사용
