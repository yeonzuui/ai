 ① Oracle에서 JDK(Java Development Kit) 다운받고 설치(17 버전)
   운영체제(윈도우/맥)에 맞는 JDK 설치
 ② 환경 변수 설정 [JAVA_HOME] : 
   윈도우 시스템 - [고급 시스템 설정] - [환경변수] - 시스템 변수 [새로 만들기] 
   - [변수 이름]에 'JAVA_HOME' 입력하고, [변수 값]에 JDK 설치경로 입력
 ③ jdk-17 내 bin 파일에는 source를 컴파일해주는 'javac.exe'와 프로그램을 실행해주는 'java.exe' 명령어가 있음
 ④ 환경 변수 설정 [Path] : bin 디렉토리 안에 있는 명령어를 어떤 위치에서도 사용할 수 있도록 경로를 추가
   시스템 변수에서 'Path' 클릭하고 [편집] - [새로 만들기] - 추가된 항목에 직접 'bin 디렉토리 경로' 를 입력 
   - [위로 이동] 해서 첫 번째 항목으로 올려주기
 ⑤ IDE(통합 개발 환경) 설치: Eclipse (설치한 JDK와 호환되는 버전인지 확인)
 ⑥ Eclipse 바로가기 아이콘 생성하고, 바탕 화면으로 이동
 ⑦ 이클립스 프로젝트 생성: [File] - [New] - [Java Project]
