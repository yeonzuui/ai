 ① DB(데이터베이스): 운영상 필요한 데이터를 중복을 최소화하여 모아 놓은 데이터 집합
 ② DBMS(데이터베이스관리시스템): 모든 응용프로그램들이 데이터베이스를 공유할 수 있도록 유지 및 관리하는 소프트웨어시스템
 ③ DBMS 기능 1) 정의: 테이블 생성, 삭제, 수정
                   2) 조작: 데이터 검색, 수정, 삭제, 조회
                   3) 제어: 무결성 유지, 보안, 복구, 병행 수행
 ④ SELECT * FROM TAB; --현 계정이 갖고 있는 테이블들
 ⑤ 날짜 표기법 셋팅(RR/MM/DD):
     ALTER SESSION SET NLS_DATE_FORMAT = "RR/MM/DD";
 ⑥ EMP 테이블 구조: 
     DESC EMP;
 ⑦ 특정 행 출력: WHERE 절(조건절) -- 비교연산자: 같다(=), 다르다(!=, ^=, <>), < <= > >=
 ⑧ 논리연산자: AND, OR, NOT / 연결연산자: ||
 ⑨ ORDER BY절
 ⑩ NVL(TO_CHAR(MGR), 'CEO')
 ⑪ DISTICNT: 중복제거
 ⑫ (NOT) BETWEEN A AND: B(A 미만, B초과)
 ⑬ IN / NOT IN: OR과 같은 개념
 ⑭ 필드 LIKE 포맷: %(0글자이상), _(한 글자)를 포함한 패턴 / 주로 검색 시 사용
 ⑮ 별칭 지정: 필드명 AS "alias"
