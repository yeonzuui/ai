 ① Sub Query : 메인 QUERY(SQL문) 안에 내포된 QUERY
 ② 비교 연산자의 오른쪽에 기술, 반드시 괄호로 둘러쌓여야 함 
 ③ 서브 쿼리는 메인 쿼리가 실행되기 이전에 한번만 실행이 됨
 ④ 서브쿼리 종류(1) 단일행 서브쿼리: 서브쿼리 결과가 단일행
 ⑤ └단일행일 때 사용가능한 연산자: =, >, <, >=, <=, !=
 ⑥ 서브쿼리 종류(2) 다중행 서브쿼리: 서브쿼리 결과가 2행 이상
 ⑦ └복수행일 때 사용가능한 연산자: IN, ALL, ANY(=SOME), EXISTS
 ⑧ ALL: 서브쿼리 결과가 모두 만족하면 참
 ⑨ ANY(=SOME): 서브쿼리 결과가 하나라도 만족하면 참
 ⑩ = ANY: IN과 동일
 ⑪ EXISTS: 서브쿼리 결과가 존재하면 참
 ⑫ F9키 누르면 선택한 부분만 드래그해서 실행 가능
